# Emma - Confluence Bot
1. Install Python latest version (3.11.5)
2. Create venv
    ```
    python3.11 -m venv .venv
    ```
3. Activate venv
    ```
    source .venv/bin/activate
    ```
4. Install packages
    ```
    python install -r requirements.txt
    ```

5. Run the application
    ```
    python -m app.main
    ```
